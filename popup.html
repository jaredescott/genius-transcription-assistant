⌨️ Keyboard Shortcuts

Ctrl + 1-5 Insert tags (Intro, Verse, Chorus, Bridge, Outro)
Ctrl + Shift + C Correct all formatting
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + Alt + Space YouTube play/pause
Ctrl + Alt + ←/→ YouTube seek ±5s
